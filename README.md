# 🚀 rodevsy-web-app

A robust and modern frontend client application built with **Next.js** for the rodevsy platform. This repository contains the code for the main user interface and client-side logic.

## ✨ Features

* **Fast Loading:** Optimized performance and routing using Next.js.
* **Responsive Design:** Ensures a seamless experience across various devices (desktop, tablet, mobile).
* **[State Management]:** Utilizes **Zustand** for scalable state management.
* **TypeScript:** Built with **TypeScript** for better type safety and code quality.
* **Styling:** Styled using **Shadcn/ui** **Tailwind**.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/)
* **Language:** TypeScript
* **Styling:** Shadcn/ui Tailwind CSS
* **API Communication:** Fetch API Hook
* **State Management:** Zustand
* **Form Handling:** React Hook Form
* **Testing:** Jest, React Testing Library
* **Linting & Formatting:** ESLint, Prettier
* **Deployment:** Vercel
* **Version Control:** Git & GitHub
* **CI/CD:** GitHub Actions
